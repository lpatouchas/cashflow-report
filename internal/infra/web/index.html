    Exclusion rules

    Exclusion rules leave matching transactions out of your totals — useful for movements that aren't real income or spending, like transfers between your own accounts that are not included in the attached account files. A transaction is excluded when it matches all of a rule's fields: match the description exactly or by contains; limit to money out (debit), money in (credit), or any; type the description to look for; and optionally a source file (leave blank to apply to every file).
